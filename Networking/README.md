# Linux 2 - Networking

## Description

These are individual labs in a group. You do the labs on your local machine and help each other.

## Prerequisites

- VirtualBox (https://www.virtualbox.org/wiki/Downloads)
- Vagrant (https://www.vagrantup.com/downloads)


## Labs

- 1 - Basic networking

- 2 - NTP

- 3 - DNS

- 4 - DHCP

- 5 - NFS

- 6 - SAMBA

- 7 - Troubleshooting

- 8 - Setup Nextcloud
