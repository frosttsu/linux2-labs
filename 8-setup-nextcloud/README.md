# Lab 8 - Setting up Nextcloud

## Objectives

Set up a functional Nextcloud Server in a VM and install Nextcloud Client on your computer (not in a VM). Connect to the Server with the Client.

## Prerequisites

CentOS 8
PHP 7.4
MariaDB
Redis (Memcache)
Apache (Web server)
Nextcloud
